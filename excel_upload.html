업로드 가이드

  • 파일 형식: Excel 파일 (.xlsx, .xls)
  • 데이터 순서: 팀 → 선수 → 대회 → 경기 → 세트점수 → 선수기록
  • 중요: 외래키 관계를 위해 순서대로 업로드해야 합니다
  • 중복 방지: 이미 존재하는 데이터는 건너뜁니다

템플릿 다운로드

데이터 입력을 위한 Excel 템플릿이 data_input_templates 폴더에 생성되어 있습니다.

  • 01_teams.xlsx - 팀 정보 템플릿
  • 02_players.xlsx - 선수 정보 템플릿
  • 03_tournaments.xlsx - 대회 정보 템플릿
  • 04_matches.xlsx - 경기 정보 템플릿
  • 05_set_scores.xlsx - 세트별 점수 템플릿
  • 06_player_match_stats.xlsx - 선수별 경기 기록 템플릿
